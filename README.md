# Pizza Store

SPA-приложение для заказа пиццы с каталогом, поиском, сортировкой, пагинацией и корзиной.

## Стек

- Frontend: React 19 (Hooks, Context API)
- State Management: Redux Toolkit & React Redux
- Routing: React Router v7
- Styling: SCSS (Sass Embedded), CSS Modules
- API Layer: Axios
- Инструменты:
- Vite — быстрая сборка проекта.
- lodash.debounce — оптимизация поисковых запросов.
- qs — парсинг и формирование строк запроса (URL query).
- react-paginate — удобная пагинация каталога.
- react-content-loader — скелетная анимация при загрузке данных.
- nanoid — генерация уникальных идентификаторов.
- @tabler/icons-react (иконки)

## Возможности

- Каталог: Фильтрация по категориям (Мясные, Вегетарианские и др.).
- Сортировка: По цене, популярности или алфавиту.
- Поиск: Быстрый поиск пицц с оптимизацией частоты запросов (debounce).
- Пагинация: Удобный переход по страницам списка товаров.
- Корзина: Добавление товаров, удаление и автоматический подсчёт итоговой суммы и количества.
- Синхронизация с URL: Сохранение фильтров и сортировки в адресной строке для возможности делиться ссылкой.
- Адаптив: Полная поддержка мобильных устройств.

## Запуск

1. Клонируйте репозиторий
2. Установите зависимости:
   npm install

3. Запустите сервер для разработки:
   npm run dev

4. Сборка проекта:
   npm run build

## Планы по доработке

- Перейти на Redux Toolkit
- Полный переход на TypeScript (типизация стейта и пропсов).
- Реализация сохранения состояния корзины в LocalStorage.
- Корзина: Добавление товаров, изменение количества, удаление и автоматический подсчёт итоговой суммы.
