# Pizza Store

Современное Single Page Application для заказа пиццы. Проект демонстрирует работу с удаленным API, сложную фильтрацию, управление глобальным состоянием и синхронизацию интерфейса с адресной строкой браузера.

## Стек

- Core: React 19 (Hooks)
- State Management: Redux Toolkit (Slices, AsyncThunk для асинхронных запросов)
- Routing: React Router v7 (Синхронизация фильтров с URL через qs)
- Styling: SCSS, CSS Modules, Адаптивная верстка
- API Layer: Axios (Интеграция с MockAPI)
- Оптимизация:
  - react-content-loader - Skeleton UI для комфортного ожидания загрузки.
  - react-paginate - удобная навигация по страницам каталога.
  - lodash.debounce - минимизация лишних запросов при поиске.
- Инструменты: Vite, nanoid, Tabler Icons.

## Ключевые архитектурные решения:

### 1. Централизованная логика запросов (Redux AsyncThunk)

Вся работа с данными вынесена из компонентов в Redux. Используется createAsyncThunk для параллельного получения списка товаров и общего количества элементов (totalCount) через Promise.all. Это делает компоненты "глухими" к реализации API и упрощает обработку состояний loading / success / error.

### 2. Умная синхронизация с URL

При изменении категории, страницы или типа сортировки, параметры автоматически "вшиваются" в URL. Это позволяет пользователю:

- Вернуться на ту же страницу при обновлении браузера.
- Поделиться ссылкой с уже примененными фильтрами.
- Корректно использовать кнопки "Назад/Вперед" в браузере.

### 3. Серверная фильтрация и поиск

Поиск реализован на стороне сервера (через API параметры), что снижает нагрузку на клиент и позволяет работать с большими объемами данных. Для предотвращения спама запросами внедрен debounce на ввод текста.

## Возможности

- Каталог: Фильтрация по категориям (Мясные, Вегетарианские и др.).
- Сортировка: По цене, популярности или алфавиту.
- Поиск пицц по названию.
- Пагинация: Постраничный вывод товаров (по 8 позиций на страницу).
- Корзина: Добавление товаров, удаление и автоматический подсчёт итоговой суммы и количества.
- Skeleton UI: Приятная визуализация процесса загрузки карточек.
- Адаптив: Полная поддержка мобильных устройств.

## Планы по доработке

- Полный переход на TypeScript (типизация стейта и пропсов).
- Реализация сохранения состояния корзины в LocalStorage.
- Изменить JSON, чтобы цены были разные для разных типов и размеров (через массив variants).

## Запуск

1. Клонируйте репозиторий:

```bash
   git clone [https://github.com/SofiaBazovaya/React-Pizza-Store.git]

```

2. Установите зависимости:

```bash
   npm install
```

3. Запустите сервер для разработки:

```bash
   npm run dev

```

4. Сборка проекта:

```bash
   npm run build
```
