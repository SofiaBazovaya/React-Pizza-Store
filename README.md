# Pizza Store

SPA-приложение для заказа пиццы с каталогом, поиском, сортировкой, пагинацией и корзиной.

## Стек

- React 18
- Vite
- React Router v6
- Redux Toolkit (Сортировка и категории)
- Context API (поиск)
- Axios + Fetch
- SCSS / CSS Modules
- lodash (debounce)
- react-paginate
- react-content-loader
- nanoid

## Возможности

- каталог пицц
- фильтрация и сортировка
- поиск
- пагинация
- корзина с подсчётом суммы
- мультистраничная навигация
- адаптивная вёрстка

## Запуск
